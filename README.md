# Pick-Place-PalletArrange-CODESYS-FactoryIO-
A pick-and-place system was developed to place 4 boxes per pallet while filling 2 pallets simultaneously. Once a pallet is full, it moves to the next production stage. The HMI includes automatic/manual modes, station color coding, step-by-step operation, and station selection.
